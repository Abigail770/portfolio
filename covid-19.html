COVID-19 Web Application:

State-Specific Projection of COVID-19 Infection


This was a project I worked on for the State Cartographer's Office along with several other graduate students. The application was requested by Professor Song Gao in the UW Geography department to display the results of his COVID-19 projection models. This was my first time helping to design and implement a web application from scratch in the workplace, and it was a great learning experience. We started out using React.js, but moved to working with vanilla Javascript, using D3 to create an interactive map and graph. This was a challenge in particular due to the large number of csv files we were working with, as well as the complicated nature of the data.
