INTERACTIVE MAPS AND GRAPHICS

Mars Rover Map

My final project for an interactive mapping class, done in collaboration with two other students. The mobile responsive web app displays the paths of three Mars rovers.

State-Specific Projection of COVID-19 Infection

A project for the WI State Cartographer's Office to display COVID-19 models.

Mitigating Flooding in Gays Mills, Wisconsin

An ArcOnline StoryMap to display a mitigation success story for Wisconsin Emergency Management.

Visualizing Patterns in Wisconsin's Tornado History

A web application created for my independent practicum project as a part of my Master's program. The goal of the application was to allow users to identify temporal patterns and trends in Wisconsin's historical torando paths.

Devil's Lake State Park

A mobile responsive app to navigate this Wisconsin State Park, explore interesting features, and submit reviews to a Carto database. This was one of two capstone projects in my Master's program.

Spatial Analysis: Nitrate Levels and Cancer Rates

A python program created with TKinter and arcpy which performs spatial analysis to analyze the relationship between nitrate levels and cancer rates in Wisconsin. This was one of two capstone projects in my Master's program.

D3 Map

One of my first web applications, created using D3 for an interactive mapping class.

Leaflet Map

My first web map application, created using Leaflet for an interactive mapping class.
